# ⚔ CommandGUI — Fabric Mod for Minecraft 1.21.1

مود Fabric لواجهة أوامر ماين كرافت الرسومية — بواسطة **has_7iq**

---

## ✨ المميزات

| الميزة | التفاصيل |
|---|---|
| 🎮 واجهة رسومية | نافذة بتصميم ماين كرافت الداكن |
| 🔍 بحث فوري | ابحث في جميع الأوامر لحظياً |
| 🗂 تصنيفات | فلترة حسب: Gamemode / Time / Weather / Player / Teleport / World / Spawn / Items |
| 📜 40+ أمر جاهز | أوامر مرتبة ومصنفة جاهزة للاستخدام |
| ⌨ اختصار لوحة مفاتيح | اضغط **G** لفتح/إغلاق الواجهة |
| 🖱 تمرير الماوس | تمرير بعجلة الفأرة داخل القائمة |
| ⚡ تنفيذ فوري | انقر على الأمر لتنفيذه مباشرة |
| 🔢 أوامر متعددة | يدعم تنفيذ أكثر من أمر بنقرة واحدة |

---

## 🎯 كيفية الاستخدام

1. ادخل إلى عالم (Creative أو Server مع صلاحيات)
2. اضغط **G** لفتح واجهة CommandGUI
3. اكتب في حقل البحث أو اختر فئة
4. انقر على الأمر المراد لتنفيذه فوراً

---

## 📦 التثبيت

```
1. ثبّت Fabric Loader 0.16.5+
2. ثبّت Fabric API 0.102.0+1.21.1
3. ضع ملف commandgui-1.0.0.jar في مجلد mods/
```

---

## 🔨 البناء (للمطورين)

```bash
./gradlew build
# الملف الناتج: build/libs/commandgui-1.0.0.jar
```

**المتطلبات:** Java 21 — Minecraft 1.21.1 — Fabric API
